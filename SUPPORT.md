# Support

Use these channels based on the type of help needed:

- Bugs: open a GitHub issue using the bug report form.
- Feature ideas: open a GitHub issue using the feature request form.
- Security vulnerabilities: follow `SECURITY.md`.
- Usage questions: start a GitHub Discussion if discussions are enabled; otherwise open a question issue.

Maintainers may not be able to provide guaranteed response times unless a separate support arrangement exists.
