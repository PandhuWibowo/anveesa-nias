# Project Summary

Anveesa Nias is an open-source database studio with a Go backend and Vue frontend.

Current public-facing project setup includes:

- MIT license.
- Open-source README and documentation index.
- Contributing, support, security, and code of conduct files.
- GitHub issue templates and pull request template.
- CI for backend tests and frontend builds.

For current contributor guidance, see:

- [Open-source project guide](OPEN_SOURCE.md)
- [Contributing guide](../CONTRIBUTING.md)
- [Security policy](../SECURITY.md)
